• We can declare variable with 'var' without intialization.

    • Ex: var x;

  • We can reassign variable with 'var'.

    • Ex: var k = 10;

    k = 12;

  • We can re-declare variable with 'var'.

    • Ex: var x = 3;

    var x = 4;

  • We can declare variable with 'let' without intialization.

    • Ex: let x;

  • We can reassign variable with 'let'.

    • Ex: let k = 10;

    k = 12;

  • We cannot re-declare variable with 'let'.
  • We cannot declare variable with 'const' without intialization.
  • We cannot reassign variable with 'const'.
  • We cannot re-declare variable with 'const'.
